# CoreSandbox
Project for learning how to implement basic mechanics within Unreal.
Learned:
  - Character movement
  - Input mapping
  - Custom locomotion logic
  - Line traces
  - Actor interaction
  - Basic inventory & equipment
  - Modular item data design
Tasks:
  - Implement sprinting, crouching, prone, parkour (vault/mantle)
  - Add stamina as an attribute (preparing for GAS)
  - Create pick-up items
  - Create an extensible inventory that supports arbitrary item types
  - Basic crafting or combining system
